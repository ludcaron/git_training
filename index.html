Aside

ceci est une zone latérale

Cette page est placée dans un dépot git qui me sert de test. Il m'a fallu dans un premier temps initialiser un repo git. puis créer les fichiers index.html et style.css.

Dans un premier temps, j'ai effectué mes premiers essais de git dans ce répertoire. j'en ai profité pour créer des branches afin de faire des commits basics et et des merges. ensuite j'ai créer un fichier conflit.txt afin dans lequel j'ai fait des modifications dans différentes branches en même temps afin de m'exercer à la résolution de conflits lors des merges.

Ensuite, j'ai créé un repository distant sur GitHub, j'ai poussé mon repo local sur github, puis j'ai récupéré ce repo dans un autre dossier de mon ordi afin de simuler un autre utilisateur. de la j'ai créé une branche ditante que j'ai poussé en remote et récupéré dans l'autre dossier. je peux maintenant continuer en m'amusant, passant d'un dossier à l'autre tout en récupérant les modifs de l'un ou de l'autre repo.

Initialisation du repository Git

Pour commencer il faut créer un nouveau dossier dans votre ordinateur. Allez dans le répertoire voulue, faite un clic droit et ouvrez git bash dans ce répertoire
